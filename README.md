# 🚀 Quiz Platform

A Full Stack Quiz Platform built using React, Node.js, Express.js, and MongoDB.

## 📌 Features

* 👤 User Registration
* 🔐 User Login Authentication
* ❓ Multiple Choice Questions
* 🔀 Random Question Order
* 🔄 Random Option Order
* ⏳ Quiz Timer
* 📊 Score Calculation
* 📈 Percentage Calculation
* 🏆 Rank System
* 🥇🥈🥉 Leaderboard
* 🚪 Logout Functionality
* 💾 MongoDB Database Integration

---

## 🛠️ Technologies Used

### Frontend

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Hema-01-hub/Quiz-platform.git
```

### Frontend Setup

```bash
cd client
npm install
npm start
```

### Backend Setup

```bash
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the server folder.

```env
MONGO_URL=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5001
```
